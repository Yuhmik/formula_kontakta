# Персонажи (мастерское)

Главный мастерский справочник по персонажам. Служит источником для трёх видов материалов:

- **Публичный список ролей** (`public/Роли.md`) – генерируется из блоков `PUB_INFO...PUB_END` и идущей перед ним информации (имя, роль, возраст).
- **Личная карточка игрока** – весь текст вне тегов `GM_INFO...GM_END`.
- **Мастерские заметки** – блоки `GM_INFO...GM_END`, не попадают ни в публичный список, ни в карточку игрока.

| Тег | Назначение |
|-----|-----------|
| `PUB_INFO` / `PUB_END` | Публичная информация – в список ролей |
| `GM_INFO` / `GM_END` | Только для мастеров: последствия поручений, кастинговые заметки, отложенные подсказки игроку |
| *(без тега)* | Личная карточка – выдаётся игроку |
