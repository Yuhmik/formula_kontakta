# Персонажи (мастерское)

Главный мастерский справочник по персонажам. Служит источником для трёх видов материалов:

- **Публичный список ролей** (`public/Роли.md`) – генерируется из блоков `PUB_INFO...PUB_END` и идущей перед ним информации (имя, роль, возраст), в конце добавляется `FOOTER_INFO...FOOTER_END`.
- **Личная карточка игрока** – весь текст вне тегов `GM_INFO...GM_END`.
- **Мастерские заметки** – блоки `GM_INFO...GM_END`, не попадают ни в публичный список, ни в карточку игрока.

См. файл `инструкции_по_генерации.md` для точных инструкций по генерации.

Все материалы из разделов "Деньги:", "Раздатка:" и "Материалы в процессе игры:" должны быть упомянуты в соответствующих разделах файла `АХЧ.md` (там написано, в каких именно разделах).

✦ Персонажи с пометками "Гендер персонажа можно инвертировать" могут по запросу игрока быть переписаны в другом гендере.
FOOTER_END
