# Формула контакта

Ролевая игра живого действия (LARP). Россия, 1835 год – инопланетяне в русской провинции.

**Армения, 12–13 сентября 2026. Коттедж, 18 игроков, 2 дня.**

---

## О чём игра

1835 год. Усадьба, липовая аллея, провинциальное общество. С неба падает нечто разумное – и прячется в местных животных, пока помещики пытаются понять, что происходит: бесовщина, болезнь или государственный заговор?

Твёрдая научная фантастика в реалистичном историческом сеттинге. Никакой мистики. Пришельцы не объясняют, кто они – они передают образы. Люди интерпретируют на свой лад.

## Структура репозитория

```
Персонажи.md          — мастерская: полные описания персонажей
Пришельцы.md          — мастерская: спецификация пришельцев
Механики.md           — мастерская: правила игры
Корреспонденция.md    — мастерская: письма и поручения
Динамика.md           — мастерская: анализ хода игры
АХЧ.md               — мастерская: материальная подготовка
Записки.md            — мастерская: утренние записки персонажей
Мастерский_лист.md    — инструмент ведущего

public/               — открытые материалы для игроков
  intro.md            — анонс
  Роли.md             — список ролей для выбора
  Атмосфера.md        — атмосфера, костюмные референсы
  Механики*.md        — правила для игроков
  characters/         — личные карточки персонажей
  letters/            — письма (шаблоны и входящие)
  exposition/         — предигровые публикации
  игротехам/          — материалы для игротехников
  ассигнации/         — игровые деньги для печати

материалы/            — исторические справки и визуальные референсы
черновики/            — рабочие черновики
```
